# Current Focus

Primary hypothesis to push this week

H1 — The hippocampal tail has stronger connections to the posterior thalamus and posterior medial cortex, compared to the hippocampal head.

Secondary hypothesis

H2 — Stronger connections (tail>head) to posterior thalamus and posterior medial cortex persist when controlling for euclidean distance, and after excluding some electrodes that touch white matter. 

---

# Main Objective This Week

Generate the **first preliminary hippocampal connectivity maps** using existing CCEP data.

Goals

0. Recreate maps generated by Lyu, et al. In Nature Neuroscience 2025, Figures 6c and 6d with data and code provided.
1. Use same methods as Goal 0 to generate tail → brain connectivity heatmap
2. Head vs tail contrast map
3. Identify strongest ROIs preferentially connected to tail

# Key Questions for the Agent Team

1. Can we recapitulate prior findings in figure 6C, 6D, supplementary table 2 and supplementary table 3 exactly (Lyu, et al.), indicating we are implementing the correct code and statistics?
2. Can we make sensible cutoffs for n=X patients, n=Y electrodes as minima for a particular ROI to be analyzed statistically? We will not make progress testing tail-occipital connections between there are maybe 1 or 2 such contact pairs. What is a good threshold to consider tail-brain region X?
2. Do posterior thalamic electrode regions show stronger connectivity with tail than head?
3. Does the PMC (posterior medial cortex) show stronger connecticity with the tail than head?

# Current Constraints

Dataset limitations

- Hippocampal tail electrodes are relatively sparse
- Some ROIs have limited partners with tail, not much coverage
- UMAP=1 was used by Lyu, et al., but we cannot use that because it destroys all our statistics.

Analysis constraints

- Must account for Euclidean distance
- Must test ipsilateral connections first
